ğŸ› TROUBLESHOOTING
================================================================================

PROBLEM 1: Port 3000 tidak bisa diakses
SOLUTION:
    # Cek firewall
    ufw status
    
    # Allow port 3000
    ufw allow 3000
    
    # Cek di Hostinger Dashboard juga
    # VPS â†’ Firewall â†’ Allow port 3000

PROBLEM 2: Bot crash terus-menerus
SOLUTION:
    # Lihat error logs
    pm2 logs whatsapp-bot
    
    # Restart bot
    pm2 restart whatsapp-bot
    
    # Jika masih error, reinstall:
    cd /root/whatsapp-bot-admin
    rm -rf node_modules
    npm install
    pm2 restart whatsapp-bot

PROBLEM 3: QR Code tidak muncul
SOLUTION:
    # Lihat logs untuk QR code di terminal
    pm2 logs whatsapp-bot
    
    # QR code mungkin muncul di logs
    # Scan dari sana

PROBLEM 4: Memory habis (Out of Memory)
SOLUTION:
    # Upgrade VPS plan di Hostinger
    # Atau optimize memory:
    pm2 start server.js --name whatsapp-bot --max-memory-restart 300M

PROBLEM 5: Session WhatsApp hilang setelah restart
SOLUTION:
    # Normal, .wwebjs_auth terhapus
    # Scan QR code lagi
    # Untuk persist, pastikan folder .wwebjs_auth tidak terhapus
